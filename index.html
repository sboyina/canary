Canary View On Github

How Canary can help you.

Are you facing this problem : You have 6 Mp3 songs (say) in a folder. But, when you open your music player library, you are finding only 3 songs in the album.

If you want all the songs in a folder to be available as part of a single album, then Canary can help you with that.

How Canary does that.

A mp3 file contains information like album title, artist, year etc.,. This information is called mp3 tags.

A Music player reads these tags and figures out the album to which the song belongs. Then, music player groups these songs by album title. This works perfectly fine when there are correct tags on the mp3 files. If these tags are not correct or corrupted, you will not find songs where they are expected. This problem often occurs with old songs.

One can correct these tags with software like JAudioTagger. We are providing an another approach to solve this problem. If you have your songs belonging to an album grouped in a folder, then using this software you can simply stamp the folder name as album title on the song. After that, your music player should show all songs in a folder as part of a single album.

Downloads

Canary-1.0.0.exe Canary-1.0.0.zip

Software Requirements.

  • Java 1.5 or higher must be present on your system.

How to run the program on Windows

  • Download exe file and run it.

How to run the program on other operating systems.

  • Download the zip file
  • Extract the contents of the zip file to a folder
  • Open the command prompt and change directory to the folder where zip is extracted.
  • Run the following command "java -jar canary-1.0.0.jar"

How to use the software.

1. Run the software as mentioned in the above sections. You will see the above screen once you run the software.
2. Click on the folder icon to select the music library folder to process.
3. By default, all sub-folders in the selected folder are also added for processing. If you don't want sub-folders to be processed, then un-check the check box.
4. If this checkbox is checked, then canary will copy the folder(in which the song is found) name as album title.
5. If this checkbox is checked, then canary will copy the file name as song title.
6. If this checkbox is checked, then canary will erase the artist and contributing artist tags on the mp3 file.
7. After making sure that you have selected appropriate options, click on start button.
8. Please wait, till you see the success messsage.
