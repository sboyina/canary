About

A mp3 file contains information like album title, artist, year etc.,. This information is called mp3 tags.

A Music player reads these tags and figures out the album to which the song belongs. Then, music player groups these songs by album title. This works perfectly fine when there are correct tags on the mp3 files. If not, you will not find songs where they are expected. This problem often occurs with old songs. One can correct these tags with software like JAudioTagger. We are providing an another approach to solve this problem. If you have your songs belonging to a album grouped in a folder and the name of the folder is same as that of the album, then using this software you can simply stamp the folder name as album title on the song.
